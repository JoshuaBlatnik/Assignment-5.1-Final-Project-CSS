# Tech Savvy: A Guide to the Latest in Technology

## Description
This web page serves as a guide to the latest trends and updates in technology. It features various sections including the latest news, reviews, tutorials, and community discussions. The design is clean and modern, ensuring an appealing user experience.

## Features
- **Smooth Scrolling:** The entire document has smooth scrolling enabled.
- **Responsive Navigation:** A sticky navigation bar that stays at the top of the page as you scroll.
- **Interactive Sections:** Each section of the main content area has a hover effect and a 'Read more' button.
- **Enhanced Footer:** The footer includes social media icons with hover effects, links to other pages, and a smooth text animation.

## File Descriptions
### index.html
The HTML file contains the structure of the webpage, including the header, main content sections, and footer. 

### style.css
The CSS file styles the webpage with the following notable features:
- **Smooth Scrolling:** `scroll-behavior: smooth;` for the entire document.
- **Footer Styling:** Enhanced styling for the footer, including social media icons with hover effects.

### script.js
The JavaScript file adds interactivity to the webpage:
- **Intersection Observer:** Adds a 'visible' class to sections as they come into view, triggering CSS animations.

## Installation and Usage
1. Ensure all files (`index.html`, `style.css`, `script.js`) are in the same directory.
2. Open `index.html` in a web browser to view the webpage.
3. The webpage will automatically apply smooth scrolling and enhanced footer styling as described.

## Customization
- To change the footer styling, modify the corresponding CSS rules in `style.css`.
- Additional interactivity can be added by modifying `script.js`.
